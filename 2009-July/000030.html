> Ich habe gestern mal mit dem Layout angefangen und muss sagen:
> Das wird nicht einfach. Der Platz auf der Platine ist zwar ausreichend
> für die Bauteile,
> aber die Tatsache das die Flash in PLCC-Fassungen stecken macht das
> Layout sehr
> kompliziert und es werden viele Duchkontaktierungen nötig sein.

Das der Platz für die Bauteile ausreicht aber trotzdem das Layout mit den
PLCC-Fassungen nicht einfach wird, haben wir schon zuvor befürchtet. Ich
hatte mir mit skoe darüber schon einmal Gedanken gemacht. Mein Vorschlag
wäre, folgende Anordnung:

+---------------------+
| +-+ +-+ +-+ +-+ +-+ |
| |U| |U| |U| |U| |U| |
| |?| |?| |?| |?| |?| |
| +-+ +-+ +-+ +-+ +-+ |
|                     |
| +-----+     +-----+ |
| | 29F |     +-----+ |
| | 040 |     |     | |
| +-----+     |     | |
|             | RAM | |
| +-----+ +-+ |     | |
| | 29F | |U| |     | |
| | 040 | |6| |     | |
| +-----+ +-+ +-----+ |
|                     |
|||||||||||||||||||||||
|||||||||||||||||||||||

Die Herausforderung wäre, die Verbindungen zwischen den beiden PLCC-Sockeln
herzustellen. Dabei müssen beiden Flächen (oben und unten) verwendet werden.
Hier sind also die Leiterbahnen beider Flächen vertikal zu legen. Daher
scheidet ein Autorouter aus. Des weiteren müssen vermutlich mind. zwei nur
Not auch drei Leiterbahnen zwischen den Pins der PLCC-Fassungen geführt
werden. Das Umsortieren der Leiterbahnen kann dann unter dem RAM bzw. in
dessen Nähe erfolgen.

|||||||||||||    | | | | | |
||0|0|0|0|0||    |o|o|o|o|o|
0|0 0 0 0 0 0   o|o|o|o|o|o|o
o 0       o o   o|o| | | |o|o
o o       o o   o|o| | | |o|o
o o o o o o o   o|o|o|o|o|o 0
  o o o o o       0 0 0 0 0

o: Pin noch nicht angeschlossen
0: Pin mit Leiterbahn
|: Leiterbahn (vertikal)

Damit wären 19 von 32 Leitungen abgedeckt. Es fehlen also noch 13 Leitungen,
also 6 auf jeder Seite. Wenn man diese noch zusätzlich zwischen den Pins
führt, sollte das aufgehen. Die Herausforderung sind dann nur die Pins auf
der linken bzw. rechten Seite der Fassung zu diesen vertikalen Verbindungen
zu routen. Da jedoch die PLCC32-Fassung quer liegt, sind da auch nur
einzelne bis max. drei Pins, welche Probleme machen.

> Ist also eher ungeeignet um die Platine mit Hausmitteln herzustellen.

Das müßte man zum Schluß mal genauer ansehen. Klar ab drei Leiterbahnen
wird's kritisch.

> Und es braucht mehr Zeit das zu layouten, ich hoffe Ihr habt es nicht
> extrem eilig ;)

Versuche mal Dein Glück. Poste evtl. Zwischenstände und Probleme und wir
helfen Dir weiter.
