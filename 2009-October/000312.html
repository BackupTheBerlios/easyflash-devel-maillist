Mir ist auf die Schnelle folgendes aufgefallen:

Beim FE werden die Summen ohne das die Seite erneut geladen wird berechnet. Offensichtlich passiert das mit JavaScript. Beim EasyFlash hast Du das ohne JavaScript gelöst. Ich persönlich finde das mit JavaScript schöner, jedoch sollte man ein Fall-Back für Browser ohne JavaScript einbauen. Dazu könnte man die Grundlage wie beim EasyFlash verwenden und zusätzlich mittels dem Event "onClick" das JavaScript einbinden.

Außerdem vermisse ich eine mögliche Übersicht, was bei den Bausätzen (alle benötigten Teile) alles dabei ist. Ein Link auf eine Stückliste wäre hier etwas transparenter.

Des weiteren fehlt eine Navigationsmöglichkeit aus den Projekten (FE oder EF) zurück zu der Übersichtsseite des Shops!
